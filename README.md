Button Counter Project

## Getting Started Locally

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## How to Build
```bash
npm run build
npm start
```
## Features Added
- Display Current Count
- Increment (+1) Button
- Decrement (-1) Button
- Reset Button
- Steps to increase and decrease amount
- Propers: initialCount and initialStep added.
- Clean layout and CSS.

## Known Issues
- Step and Count values do not presist after page reload.
